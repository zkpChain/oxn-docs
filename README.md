## 关于 OXN

OXN 是一条机密 EVM 测试链：EVM 兼容、Solidity 合约直接复用，**默认对交易 calldata 和 view-call 自动加密**，
链上观察者看不到调用方法和参数。

公开入口：

- RPC：<https://rpc.bout.network>
- Wallet：<https://wallet.bout.network/>
- Explorer：<https://explorer.bout.network/>
- Faucet：<https://faucet.bout.network/>
